# Top 10 Animes

## Sobre o Projeto

Este é um projeto de front-end que exibe uma lista interativa dos 10 melhores animes. O objetivo é fornecer uma interface de usuário simples e agradável para explorar informações sobre essas obras, incluindo a visualização de clipes de vídeo.

## Usabilidade e Funcionalidades

O projeto foi desenvolvido com foco na simplicidade e facilidade de uso. As principais funcionalidades incluem:

*   **Navegação Intuitiva:** O usuário pode navegar facilmente pela lista de animes.
*   **Informações Detalhadas:** Ao interagir com um anime da lista, são exibidos detalhes como descrição, criador e data de lançamento.
*   **Visualização de Vídeos:** Cada anime possui um clipe de vídeo associado que pode ser reproduzido diretamente na página.
*   **Design Responsivo:** A aplicação se adapta a diferentes tamanhos de tela, proporcionando uma boa experiência tanto em desktops quanto em dispositivos móveis.

## Configuração dos Vídeos

Antes de utilizar o projeto, é necessário baixar os vídeos dos animes.

1.  Acesse o arquivo `assets.md` para encontrar o link de download.
2.  Acesse o link do Google Drive, baixe os vídeos.
3.  Na raiz do projeto, crie uma nova pasta chamada `assets`.
4.  Coloque os vídeos baixados dentro da pasta `assets`.

## Como Utilizar

1.  Abra o arquivo `index.html` em seu navegador.
2.  Você verá a lista dos 10 animes.
3.  Clique ou passe o mouse sobre um anime para ver mais detalhes.
4.  Clique no botão de play para assistir ao vídeo correspondente.

## Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript

## Como Executar o Projeto

Para executar este projeto localmente, siga os passos abaixo:

1.  Clone o repositório (se estiver em um).
2.  Navegue até o diretório do projeto.
3.  Abra o arquivo `index.html` no seu navegador de preferência.
